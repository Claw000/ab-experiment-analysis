# AB实验分析检查清单

## 分析前确认

- [ ] 确认每个文件对应什么指标
- [ ] 确认分组含义（哪个是对照、哪些是实验）
- [ ] 确认稳定期范围
- [ ] 确认实验各组改了什么
- [ ] 确认指标单位（分 vs 元）
- [ ] **确认护栏指标是什么**（不同实验不同，不要默认）

## 分析中检查

- [ ] 流量分配是否均匀（各组活跃用户数）
- [ ] 稳定期数据天数 ≥ 4天
- [ ] 异常值检测（IQR法）
- [ ] 去异常值后是否结论反转
- [ ] 指标按归属分类（权益包 / 会员 / 综合）
- [ ] 综合收入 = 会员 + 权益包（注意单位转换）

## 输出检查

- [ ] 汇总表包含所有指标
- [ ] 每个指标标注 p值 和 显著性
- [ ] 异常值单独说明
- [ ] 护栏指标（会员开通率）单独标注
- [ ] 综合结论包含明确建议
- [ ] 建议按实验目的评价（体验目标 vs 商业目标）

## 常见错误

| 错误 | 后果 | 预防 |
|------|------|------|
| 权益包金额(分)和会员金额(元)直接相加 | 综合收入算错100倍 | 统一转换为元 |
| 用进组期数据算均值 | 稀释效果 | 只用稳定期 |
| 异常值不处理 | 一笔大单决定结论 | IQR检测+两版对比 |
| 只看收入不看体验 | 片面结论 | 分维度呈现 |
| 不问实验改了什么 | 无法判断因果 | 分析前必问 |
| 分组搞反（-1当对照） | 所有结论反了 | 确认-1是大盘 |
